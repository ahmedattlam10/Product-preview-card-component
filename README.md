#  Product preview card component

![Project Screenshot](./images/Screenshot%202026-05-06%20162924.png)

A responsive product preview card built with HTML and CSS.

## Overview

This project is a solution to the Frontend Mentor challenge.  
The goal was to build a responsive product card that works well on both mobile and desktop screens.


## Built With

- Semantic HTML5
- CSS custom properties
- Flexbox
- Mobile-first workflow

## What I Learned

- Using the `<picture>` element for responsive images
- Building responsive layouts with Flexbox
- Improving spacing and alignment using CSS
